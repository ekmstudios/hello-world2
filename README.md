# Markdown Basic Syntax
## Heading
` # H1 `
` ## H2 `
` ### H3 `

## Bold Text
` **bold text** `

## Italic
` *italicized text* `

## Blockquote
` > blockquote `

## Ordered List
1. First item
2. Second item
3. Third item

## Unordered List
- First item
- Second item
- Third item

## Code
``code``

## Horizontal Rule
` --- `

## Link
` [title](https://www.example.com) `

## Image
` ![image from my site](image.jpg) `
