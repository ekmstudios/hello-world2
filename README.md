# Markdown Basic Syntax
## Heading
` # H1 `<br />
` ## H2 `<br />
` ### H3 `<br />

## Bold Text
` **bold text** `

## Italic
` *italicized text* `

## Blockquote
` > blockquote `

## Ordered List
1. First item
2. Second item
3. Third item

## Unordered List
- First item
- Second item
- Third item

## Code
`code`

## Horizontal Rule
` --- `

## Link
` [title](https://www.example.com) `

## Link
[Heading](#Heading)

## Image
` ![alt text](image.jpg) `
